# Инструкция: Добавление Yandex токена

## Когда получишь токен от владельца магазина

### 1. Добавь токен на сервер

```bash
ssh root@89.223.125.212
nano /root/m2-middleware/.env
```

Найди строку:
```
YANDEX_TOKEN=
```

Вставь токен:
```
YANDEX_TOKEN=твой_токен_здесь
```

Сохрани: `Ctrl+O`, `Enter`, `Ctrl+X`

### 2. Перезапусти сервер

```bash
pm2 restart m2-middleware
```

### 3. Проверь что всё работает

```bash
pm2 logs m2-middleware --lines 50
```

Должны увидеть:
- ✅ "Сервер запущен на порту 3001"
- ✅ "Маппинги загружены: 101"
- ✅ "Cron job запущен: синхронизация остатков"

### 4. Проверь синхронизацию

Через 10-15 минут проверь логи:
```bash
tail -100 /root/m2-middleware/logs/combined.log | grep stock
```

Должны увидеть успешную синхронизацию остатков.

## Быстрая команда (всё в одной строке)

Или используй готовый скрипт из локального проекта:

```bash
# Из локального проекта
bash update-yandex-token.sh ВАШ_ТОКЕН_ЗДЕСЬ
```

## Что проверить после добавления токена

1. Сервер запустился без ошибок
2. Синхронизация остатков работает (каждые 10 минут)
3. Polling заказов работает (каждые 5 минут)
4. Webhook от МойСклад принимается

## Проблемы?

Если что-то не работает, запусти:
```bash
bash check-stock-sync.sh
```
